## Установка

1. Установите ```pip install -r requirements.txt```
2. Запустите ```app.py```
3. Откройте браузер и перейдите по адресу

## Использование

### Регистрация
1. Перейдите на страницу "Регистрация"
2. Заполните форму (имя пользователя, email, пароль)
3. После регистрации вы будете перенаправлены на страницу входа

### Просмотр товаров
- На главной странице отображаются все товары
- Используйте поиск для поиска товаров по названию или описанию
- Фильтруйте товары по категориям

### Корзина
- Нажмите "В корзину" на любом товаре
- Перейдите в корзину для просмотра выбранных товаров
- Измените количество или удалите товары
- Оформите заказ

### Избранное
- Нажмите на иконку ❤️ на товаре, чтобы добавить его в избранное
- Просмотрите все избранные товары на странице "Избранное"

### Отзывы
- Откройте страницу товара
- Оставьте отзыв с оценкой (1-5 звезд) и комментарием
- Просмотрите отзывы других пользователей
- Удалите свои отзывы при необходимости

## Технологии

- **Flask** - веб-фреймворк
- **SQLite** - база данных
- **Werkzeug** - хеширование паролей
- **Jinja2** - шаблонизатор
- **HTML/CSS** - интерфейс (без JavaScript)

## База данных

При первом запуске автоматически создается база данных `store.db` с таблицами:
- `users` - пользователи
- `products` - товары
- `cart` - корзина
- `favorites` - избранное
- `reviews` - отзывы

Также автоматически добавляются 6 тестовых товаров.

## Безопасность

- Пароли хранятся в хешированном виде (Werkzeug)
- Защита от SQL-инъекций через параметризованные запросы
- Проверка авторизации для защищенных страниц
- Валидация данных на сервере

## Примечания

- Это демо-приложение для обучения

  - Использовать PostgreSQL или MySQL
  - Добавить обработку ошибок
  - Реализовать систему платежей
